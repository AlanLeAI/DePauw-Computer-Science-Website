Data Type in Computer Science

In computer science and computer programming, a data type (or simply type) is a collection or grouping of data values, usually specified by a set of possible values, a set of allowed operations on these values, and/or a representation of these values as machine types. A data type specification in a program constrains the possible values that an expression, such as a variable or a function call, might take. On literal data, it tells the compiler or interpreter how the programmer intends to use the data.

Condition Operator and Mathematical Operator

Condition operators are symbols used in programming languages to evaluate a condition and determine the flow of execution of a program. Mathematical operators are symbols used in programming languages to perform arithmetic operations.
In Java, some commonly used condition operators are:

  • "==": check if two variables are equal
  • "!=": check if two variables are different
  • ">": check if the left variable is greater than the right variable
  • "<": check if the left variable is smaller than the right variable
  • ">=": check if the left variable is greater than or equal the right variable
  • "<=": check if the left variable is smaller than or equal the right variable

    • Some commonly used Mathematical Operators are:


  • "+": Addition
  • "-": Subtraction
  • "*": Multiplication
  • "/": Division
  • "%": Modulo

    Indexing Information

    In an array, an index starts at 0 and counts up from there.
    Say that you declare an array called arr with values 2, 4, 6, 8
    Declaring that array looks like: arr[] = [2, 4, 6, 8]

    The value 2 is at index 0, the value 4 is at index 1, and so on.

    Typically, programmers use the letter 'i' to show the index in a loop.

    Example: for (int i=0, i < arr.length, i++)

  • int i=0 initializes the variable i

  • i < arr.length is the condition. Here the loop will run as long as the index is less than the length of the array. We use < instead of <= because the last index in the array will always be one less than the ength of the array since indexing starts at zero.
  • i++ increments the index after everything inside the loop has executed.

    • Finding i retrieves the index.
    Finding arr[i] retrieves the value associate with that index.
    For example, using the array from above, if i = 2, arr[i] equals 6.
