Queue

A queue is a linear data structure that follows the First-In-First-Out (FIFO) principle.

Operators of Queue and Runtime

Operator Description Runtime
enqueue(item) Adds an item to the end of the queue. O(1)
dequeue() Removes and returns the item from the front of the queue. O(1)
peek() Returns the item at the front of the queue without removing it. O(1)
isEmpty() Checks if the queue is empty. O(1)
size() Returns the number of items in the queue. O(1)
